# 👋 Hello, I’m Yoriiis

Front-End Architect & Open Source creator since 2011.  
I design lightweight, scalable, and maintainable front-end systems, and develop libraries, plugins, and mini-frameworks that simplify team workflows and enhance developer experience.

My open source projects are used by **millions of users every month** via npm, CDN, Figma, and Chrome Web Store, with a focus on **performance, simplicity, and technical consistency**.
